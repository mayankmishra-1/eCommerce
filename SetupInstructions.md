# Setup Instructions

## Backend Setup
1. Clone the repo
2. Install dependencies(npm i)
3. Start server(npm start)


## Frontend Setup
1. Go to ecommerce-frontend folder
2. Start frontend(npm start)
3. Access at `http://localhost:3000`

## Demo Flow
- Use **Role Switcher** to select ADMIN or USER
- ADMIN: Add products, update stock, update order status
- USER: Add to cart, checkout, cancel orders
